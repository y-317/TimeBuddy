# 用語集

| 用語 | 定義 |
|-----|------|
| タイマー | 設定した時間からカウントダウンし、0になったら通知を発行する機能 |
| アラーム | 指定した時刻になったら通知を発行する機能 |
| ストップウォッチ | 開始からの経過時間を計測する機能 |
| カウントダウン | 設定した時間から0に向かって時間を減らしていくこと |
| ラップタイム | ストップウォッチで記録した、ある時点での経過時間 |
| スプリットタイム | 前回のラップタイムから次のラップタイムまでの経過時間 |
| モード | タイマー、アラーム、ストップウォッチのいずれかの機能状態 |
| バックグラウンド動作 | 他のモードに切り替えても、機能が継続して動作すること |
| ローカルストレージ | ブラウザにデータを保存する仕組み（localStorage API） |
| レスポンシブデザイン | 画面サイズに応じてレイアウトを最適化するデザイン手法 |
| ダークモード | 背景を黒系、テキストを白系にした配色モード |
| ライトモード | 背景を白系、テキストを黒系にした配色モード |
| ブレークポイント | レスポンシブデザインにおいて、レイアウトが切り替わる画面幅の境界値 |
| プリセット | あらかじめ設定された値。タイマーでよく使う時間（1分、5分等）を素早く設定する機能 |
